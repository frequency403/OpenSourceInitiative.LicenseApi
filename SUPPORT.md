# Support

If you need help with using OpenSourceInitiative.LicenseApi:

- First, check the [README](README.md) for installation and usage.
- Search existing [issues](https://github.com/frequency403/OpenSourceInitiative.LicenseApi/issues).
- If your question isn’t answered, open a new issue with details (environment, .NET version, steps to reproduce).

Please do NOT disclose security issues publicly — see our [Security Policy](SECURITY.md).
